# Setup `kubeadm` based Kuberenetes Cluster (local) using `Vagrant` - Win 11

- Install Oracle VM VirtualBox
- Install and Configure Vagrant
- Create and Run Virtual Machines using Vagrant
- Connect to VM and Run commands
- Other Vagrant commands (operations)

### Step-XX: Download and Install `Vagrant`

- You can download the vagrant binary from https://developer.hashicorp.com/vagrant/install and then install.

- After the installation completes, run the following command to verify it:

```
$ vagrant --version
OR
$ vagrant -v
```

### Step-XX: Select the VM Provider (VirtualBox/Hyper-V/Docker)

- You can download the Oracle VM VirtulBox from https://www.virtualbox.org/wiki/Downloads and then install it.

- After the installation completes, navigate to Start menu >> _Oracle VirtualBox_ to launch the app.
